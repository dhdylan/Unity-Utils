# DRS Core
DRS Core is a collection of general purpose utility scripts that I often use when working inside unity.
The original idea was that I always found myself rewriting (or importing from another existing project) the same high-level systems for every Unity project I was working on.

Eventually I said "Hey, I could just make a "template" Unity project that has this stuff in it already, and just use that to create new projects with.

From there I eventually said "Hey, I could just make this a Unity package and open-source it on Github."

My plan is to continue to use and extend this repo into a sort of developement framework where all of the high level architecture that I typically use - things like scene management, asset streaming, data persistence, high-level application flow - already exist and it is up to the developer to just build the thing they want to make instead of writing a global event system for the 27th time.
# Installation
### ***NOTE: This package currently has a dependency that must be manually installed***
Make sure to install [this](https://github.com/mackysoft/Unity-SerializeReferenceExtensions/tree/main) package first before installing DRS Core.


Installing this package can be done by Opening Package Manager -> "Add from git URL" and paste this into the git url field:
`https://github.com/dhdylan/DRS-Core.git`
