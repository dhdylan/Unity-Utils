# DRS Core
DRS Core is a collection of general purpose utility scripts that I often use when working inside unity.
# Installation
### ***NOTE: This package currently has a dependency that must be manually installed***
Make sure to install [this](https://github.com/mackysoft/Unity-SerializeReferenceExtensions/tree/main) package firtst before installing DRS Core.


Installing this package can be done by Opening Package Manager -> "Add from git URL" and paste this into the git url field:
`https://github.com/dhdylan/DRS-Core.git`
